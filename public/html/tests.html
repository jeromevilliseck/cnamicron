Methode3 : ciblage de plusieurs ou une balise (par son indice) sur les attributs HTML class

fichier HTML
<p class="java">un paragraphe avec un attribut class java, cet attribut en Js vaut ("java")[0]</p>
<p class="java">un autre paragraphe dont la balise à un attribut class java, cet attribut en Js vaut ("java")[1]</p>
<button onclick="nomMethode()">cliquez pour activer les changements sur les balises avec attribut class</button>
fichier JS 
    
    function nomMethode3(){
        var test = document.getElementsByClassName("java");
        for (var i = 0; i < test.length; i++) { //On cible tous les attributs class
            test[i].style.color = "green";
        }
        test[1].style.backgroundColor = "orange"; //On cible QUE l'attribut class d'indice 1
    }

ATTENTION : les attributs class se manipulent via un tableau scalaire, les class ont donc un indice, obligatoire à mentionner pour se manipuler en javacript.
