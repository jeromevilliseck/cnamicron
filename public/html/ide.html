Développement Web

Awardspace, Alwaysdata (Hebergement web)

Hébergement Awardspace

Cette rubrique va vous aider à mettre en place un hebergement web comprenant :

  • la gestion et le transfert des fichiers par protocole ftp (le protocole sftp (SSL) necessite de payer)
  • l'hebergement à distance gratuit de fichiers via un espace de 1GO sur un serveur mutualisé
  • une base de données gratuite (limitée à 99tables)
  • un accès SQL via le client MySQL (uniquement via phpmyadmin -> bloquage de l'hebergeur des softwares de SGDBR externes pour sécurité)

Le tout gratuitement et en acceptant la syntaxe des requetes SQL communes.

Hébergement Alwaysdata

Alwaysdata va vous permettre lui la meme chose mais va accepter le protocole SSL. Par contre attention la syntaxe pour les requetes SQL sera plus stricte, notamment si vous suivez mon cours SQL vous aller avoir des problèmes d'erreur par rapport à la casse ou au virgules encadrant les tables et attributs. Toutefois alwaysdata va permettre de travailler pas seulement sur MySQL, mais aussi avec les SGBD suivants :

  • PostgreSQL
  • MongoDB
  • CouchDB

Eclipse : Manipulation de l'outil, arborescence

Vous pouvez suivre les guides Objis de l'étape 4 à l'étape 8 pour vous entrainer à manipuler l'outil. Pour l'arborescence d'un projet java retenez ceci

  • Un projet java à travers le "package explorer" d'eclipse comprend un dossier src, qui comprend des dossier dits packages. Ces dossiers dits packages comprennent les fichiers .class qui comprennent les Classes java.
  • Généralement on crée 5 packages standards : controller (qui comprends la Classe principale Main.java), model (qui comprends les autres Classes pour héritage etc..), util, view, dao.
  • Les noms de packages en minuscule, les noms des classes toujours avec une Majuscule au début.
  • Le workspace est le dossier pointé ou sont regroupés tous les projets java du développeur
  • Le dossier bin comprend l'arborescence identique et les memes fichiers que le dossier src mais avec une extension .class et non .java
  • les fichiers .class contiennent le bytecode intermédiaire executé par le processeur du "liseur"
  • les fichiers .java contiennent le code source java éditable directement par le développeur
